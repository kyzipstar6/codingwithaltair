Bildschirmrekorder mit Java und ffmpeg

Hinzugefügt ab Feb 24, 2026 | Category: ffmpeg

Zuerst installieren bzw. laden Sie ffmpeg

Dann passen Sie den folgenden command an:

bildschirmaufnehmen.bat:

cd C:\Optional\Verzeichnis\zur\ausgabe

 				ffmpeg -f gdigrab -framerate 30 -i desktop -f dshow -i audio="Microphone (High Definition Audio Device)" 
					-c:v libx264 -preset ultrafast -c:a aac -b:a 128k -pix_fmt yuv420p -movflags +faststart output.mp4

Ersetzen Sie "Microphone (High Definition Audio Device)" mit der Name ihres audiogerät oder loschen sie "audio=...", falls man kein audio aufnehmen werden willt.

Building exes of JavaFX applications with JPackage & Candle

Posted on Feb 4, 2026 | Category: Jpackage

Install first Wix Toolset

Then fill the code bellow and save it as a .ps1 file or copy-paste it at powershell

Jpack.ps1:

Set-Location "C:\Path\to\your\aplication"

 jpackage `
   --input "C:\Path\to\your\aplication" `
   --name "My App" `
   --main-jar "plotter.jar" `
   --main-class "myapplicationpackage.MyAppMainClass" `
   --type exe `
   --dest "C:\Path\to\your\exe\output\other\than\input" `
   --module-path "C:\Path\to\your\javafx\libraries" `
   --add-modules javafx.controls,javafx.base,javafx.fxml `
   --win-menu `
   --win-dir-chooser `
 	--win-console `
 --java-options "-cp app/build/bin" `  
   --java-options "-Dprism.order=sw" `
   --java-options "-Dprism.verbose=true" `
   --java-options "-Djavafx.verbose=true"

If it does not work, try adding this at your public void main(String args []) of your main class:


       public static void main(String args[]) {
		System.setProperty("prism.order", "d3d,sw");
	    
		launch(args);
		}

private void applyTheme(Scene scene) {
			// JavaFX stylesheets must be loaded via URL; we write the CSS to a temp file at
			// runtime.
			final String css = """

									.chart { -fx-background-color: #cbd5e1; }
									.chart-plot-background {

											-fx-background-insets: 0;
											-fx-background-radius: 12;
									}
									.default-color0.chart-series-line { -fx-stroke: #dae97a; }
									.default-color1.chart-series-line { -fx-stroke: #8cedf0; }
									.default-color2.chart-series-line { -fx-stroke: #f08ceb; }
									.default-color2.chart-series-line { -fx-stroke: #8af2b2; }

							""";

			try {
					Path tmp = Files.createTempFile("theme", ".css");
					Files.writeString(tmp, css, StandardCharsets.UTF_8);
					tmp.toFile().deleteOnExit();
					scene.getStylesheets().add(tmp.toUri().toString());
			} catch (IOException ex) {
					// If the stylesheet fails, keep the app usable (fall back to default JavaFX
					// look).
					ex.printStackTrace();
			}
	}
